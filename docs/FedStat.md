
### Структура данных ЕМИСС

- `indicators.csv`
  * Список кодов показателей
  * Формат: CSV с единственным столбцом `code`.

- `excel/{code}.xls`
  * Оригинальные Excel-файлы, скачанные для каждого показателя.

- `metadata/{code}.json`
  * Метаданные отдельных показателей.
  * Формат: JSON, содержит `code`, `name` и свойства из листа "Паспорт".

- `metadata.jsonl`
  * Аггрегированный JSONL-файл со всеми метаданными показателей.
  * Каждая строка — отдельный JSON-объект.

- `dicts.csv`
  * CSV-словарь уникальных названий колонок, найденных в всех `csv.gz` файлах.
  * Формат: одна колонка `name`.

- `parquet/{code}.parquet`
  * Паркет-версии файлов `csv/{code}.csv.gz`.

- `clean_jsonl/{code}.jsonl.gz`
  * Очищенные, нормализованные JSONL-файлы, сжаты в gzip.
